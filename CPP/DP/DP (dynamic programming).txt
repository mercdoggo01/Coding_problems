

DP (DYNAMIC PROGRAMMING)

Pre requisite : Recursion (intermediate level knowledge)

Technique:          1.) Think brute force recursion

                                2.) Make table as necessary
                                         (i.e assume default 10^5 size)

                                3.) For the recursive function 
                                       take only those values whose state changes.
                               
                                4.) dry run brute force before testing the dp 
                                       implementation.
